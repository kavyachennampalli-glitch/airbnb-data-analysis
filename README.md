# Airbnb Open Data Analysis

## Project Overview
This project performs Exploratory Data Analysis (EDA) on the Airbnb Open Data dataset using Python. The project focuses on cleaning the data, exploring patterns, and creating visualizations to generate meaningful insights.

## Tools & Technologies
- Python
- Jupyter Notebook
- Pandas
- NumPy
- Matplotlib
- Seaborn

## Project Workflow
- Data Collection
- Data Cleaning
- Handling Missing Values
- Removing Duplicates
- Exploratory Data Analysis (EDA)
- Data Visualization
- Business Insights

## Key Insights
- Analyzed Airbnb listing prices.
- Compared room types.
- Explored neighborhood trends.
- Identified correlations between variables.
- Created visualizations using Matplotlib and Seaborn.
